# Coffee Shop Bingo Design Spec

## Theme Overview
Transform the Bingo Mixer into a cozy coffee shop experience with warm, inviting colors and coffee shop elements.

## Color Palette
- Primary: Warm coffee browns (#8B4513, #A0522D)
- Secondary: Creams and beiges (#F5F5DC, #FFF8DC)
- Accents: Soft amber and gold (#DAA520, #FFD700)
- Background: Light parchment (#FAF0E6)

## Key Elements
- Wooden tables and coasters
- Coffee-stained napkins
- Steam from mugs
- Relaxed, welcoming atmosphere
- Subtle animations for warmth

## Component Transformations
- StartScreen: Coffee shop entrance/welcome with steam animations
- GameScreen: Cozy table setting with wooden header
- BingoBoard: Coffee-stained napkin with stain effects
- BingoSquare: Coffee-themed items with steam on marked squares
- BingoModal: Celebration with coffee/tea elements

## Typography
- Heading: 'Dancing Script' cursive font
- Body: 'Crimson Text' serif font
- Applied via CSS variables in @theme

## Animations
- Gentle steam rising animations on start screen
- Hover effects with scale transforms
- Pulse animations for steam
- Bounce animation on bingo modal
